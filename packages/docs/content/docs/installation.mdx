---
title: Installation
description: Getting started
---

Install the `@unriddle-ai/lector` package with your favourite package manager:

```shell
pnpm add @unriddle-ai/lector
```

```shell
npm install @unriddle-ai/lector
```

```shell
yarn add @unriddle-ai/lector
```

```shell
bun add @unriddle-ai/lector
```
