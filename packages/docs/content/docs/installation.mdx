---
title: Installation
description: Getting started
---

Install the `@unriddle-ai/pdf-viewer` package with your favourite package manager:

```shell
pnpm add @unriddle-ai/pdf-viewer
```

```shell
npm install @unriddle-ai/pdf-viewer
```

```shell
yarn add @unriddle-ai/pdf-viewer
```

```shell
bun add @unriddle-ai/pdf-viewer
```
