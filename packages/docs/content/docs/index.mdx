# Welcome to Lector

Lector is a powerful, composable PDF viewer toolkit for React applications, powered by PDF.js. Build feature-rich PDF viewing experiences with full control over the UI and functionality.

## Quick Start

Get started with Lector in three simple steps:

1. Install the package:

```bash
npm install @unriddle-ai/lector pdfjs-dist
```

2. Import and use the components:

```tsx
import { Root, Pages, Page, CanvasLayer, TextLayer } from "@unriddle-ai/lector";

function PDFViewer() {
  return (
    <Root fileURL='/sample.pdf'>
      <Pages>
        <Page>
          <CanvasLayer />
          <TextLayer />
        </Page>
      </Pages>
    </Root>
  );
}
```
